# Full Calculator - Python FastAPI

## Como usar

1. **Clone o repositório**
   ```sh
   git clone https://github.com/lucasmartinssw/API-SuporteTI
    ```
2. **Instale as dependências**
```sh
pip install -r "requirements.txt"
```

3. **Rode o projeto**
```sh
python -m uvicorn app.main:app --reload
```
